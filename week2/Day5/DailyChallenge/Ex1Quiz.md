Exercise 1: Quizz

Answer the following questions:

    What is a class?
        A class is a category of objects with its own properties, methods, and the like we can set.
    What is an instance?
        An instance is an object within a class.
    What is encapsulation?
        Encapsulation is the process of restricting access to methods and variables to prevent data from direct modificaiton.
    What is abstraction?
        Abstraction is the process of hiding the complex internal details of a system and exposing only the essential features to the user.
    What is inheritance?
        Inheritance is how a class can be defined based on another class and inherit its properties, methods, etc.
    What is multiple inheritance?
        Multiple inheritance is when a class inherits from two different classes.
    What is polymorphism?
        Polymorphism is when different (or related) classes use the same names for their functions.
    What is method resolution order or MRO?
        MRO is the order in which Python looks for a method or attribute in a class hierarchy.
